# Sri Lanka Export Credit Insurance Corporation (SLECIC)

This project is a web application for the Sri Lanka Export Credit Insurance Corporation (SLECIC). It aims to support and manage export credit insurance services, providing exporters with risk protection and financial security.

## Project Structure
- `index.php` — Main entry point of the application
- `assets/` — Static files (CSS, images)
- `config/` — Configuration files (e.g., database connection)
- `includes/` — Reusable PHP components

## Getting Started
1. Clone or download the repository.
2. Set up your web server (e.g., XAMPP) and place the project in the web root directory.
3. Configure the database in `config/db.php`.
4. Access the application via your browser at `http://localhost/SLECIC/`.
